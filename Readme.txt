# Shopping Website

This is a simple shopping website built with React and Next.js, Tailwind css and css. It allows users to view a list of products, add them to the cart, and manage the cart by incrementing, decrementing, or removing items.

## Features

- View a list of products with details such as title, description, and price.
- Add products to the cart.
- Increment or decrement the quantity of items in the cart.
- Remove items from the cart.
- Responsive design for a seamless experience on different devices.
